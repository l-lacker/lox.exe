# lox.exe
программа, улучшающая качество жизни — теперь опенсорс!

## установка
1. скачайте или склонируйте репозиторий на компьютер
2. перенесите папку bundles сюда:
```
C:\Users\%username%\DevelNextLibrary\
```
3. вы великолепны! проект можно открывать через DN
